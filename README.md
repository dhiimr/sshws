# sshws
SSH over Websocket

Connect your domain to Cloudflare
add in Record DNS A domain TO IP
set SSL/TLS to Full (strict)

default Port WS 8080
default Port WS SSL 443
default Port SSH 22
default Port SSH SSL 143
